**UNITS**

    -units/ - получить все (GET)
    -units/ - добавить (POST)
    -units/:id - получить по id (GET)
    -units/:id - обновить по id (PUT)
    -units/:id - удалить по id (DELETE)

**FILE_STORAGE**

    -fs/ - загрузить (POST)
    -fs/:name - получить файл по имени (GET)
    -fs/:name - удалить файл по имени (DELETE)
